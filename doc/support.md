# Support

## Releases

Migration Manager has frequent releases with feature updates as well as security updates and bugfixes.
Support for a release normally lasts until the next release is published.

## Bug reports

Issues, bug reports, and feature requests can be reported at [`https://github.com/FuturFusion/migration-manager/issues`](https://github.com/FuturFusion/migration-manager/issues)

## Community support

Community support is available at [`https://github.com/FuturFusion/migration-manager/discussions`](https://github.com/FuturFusion/migration-manager/discussions)

## Commercial support

Commercial support is available as part of FuturFusion Cloud at [FuturFusion](https://futurfusion.io)
